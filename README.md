# NOT LIKE US — The Rap War That Changed Everything

An interactive scrollytelling website documenting the 2024–2026 cultural phenomenon of the Kendrick Lamar vs. Drake rap feud.

## 🔗 Live Site
[View the website](https://YOUR-USERNAME.github.io/REPO-NAME/)

## What's Inside
- **Origins** — Toronto vs. Compton, the environmental dichotomy
- **The War of Words** — Full timeline of all 10 diss tracks with embedded audio/video
- **Lyrical Autopsy** — Click-to-compare interactive bar-by-bar analysis
- **Face-Off** — Side-by-side breakdown of Family Matters vs. Not Like Us
- **Who Stood Where** — The full alliance network with artist profiles
- **Global Impact** — Streaming data, polling numbers, Google Trends
- **The Grammy Sweep** — 5 awards, one diss track
- **Super Bowl LIX** — 133.5M viewers, the final blow
- **Legal Fallout** — Drake vs. UMG, the First Amendment battle
- **Verdict** — Interactive poll

## Built With
- Vanilla HTML, CSS, JavaScript — fully self-contained, no frameworks
- YouTube embed API
- Twitter & Facebook embed widgets
- Google Fonts (Bebas Neue, Playfair Display, IBM Plex Mono, Barlow Condensed)
